# 사용자 추가 방법

## server.js 파일에서 사용자 추가하기

[server.js](server.js) 파일의 33-40번째 줄에 있는 `users` 객체에 사용자를 추가하세요:

```javascript
const users = {
    'user1@company.com': '1234',
    'user2@company.com': '5678',
    'admin@company.com': '0000',
    // 새로운 사용자를 아래에 추가하세요
    'newuser@company.com': '1111'
};
```

## 형식
- **이메일 주소**: 'email@company.com'
- **PIN 번호**: '1234' (4자리 숫자)

## 예제
```javascript
const users = {
    'hong@company.com': '1234',
    'kim@company.com': '5678',
    'park@company.com': '9999'
};
```

## 서버 재시작
사용자를 추가한 후 반드시 서버를 재시작해야 합니다:

```bash
# 1. Ctrl + C로 현재 서버 중지
# 2. 서버 재시작
npm start
```

## 보안 주의사항
실제 운영 환경에서는 다음을 권장합니다:
1. 데이터베이스 사용 (MySQL, MongoDB 등)
2. 비밀번호 해싱 (bcrypt 등)
3. JWT 토큰 인증
4. HTTPS 사용
